Hexo 的搭建

Hexo 的搭建还是蛮简单的，命令行几行搞定, 参考Hexo 文档,这里就不细讲了

多台机器上写 Blog 怎么办？

Hexo 发布主要是打包静态文件然后部署, 但是如果我们是在不同的机器上写 Blog 的话，那就很操蛋了, 可能又要重新搭建…, 这样的话为了  多台机器上都能流利地写吹逼，一般都会将 hexo 的文件放到 github 上，在别的机器上直接 clone 下来，
 然后将可以继续愉快地吹逼了～
